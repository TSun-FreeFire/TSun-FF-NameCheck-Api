FreeFire BanCheck API is a lightweight Flask-based REST API that retrieves Free Fire player information with ban status. Designed for speed, precision, and integration with custom dashboards or moderation tools.

Version 1.0.0
Base URL https://bancheck.tsunstudio.pw

Response Fields

Field Type Description
nickname string Player's in-game nickname
uid string Player's unique identifier
AccountLevel integer Player's account level
region string Player's region (PK, BD, IND, NA, SG)
AccountLastLogin string Last login date (YYYY-MM-DD format)
status string API request status
is_banned boolean Whether the account is banned
Last_Login string Human-readable time since last login
credits string API credits

ğŸ’¡ Examples

URL

bash 
https://bancheck.tsunstudio.pw/bancheck?uid=2513698016

Python

python 
import requests

uid = "2513698016"
response = requests.get(f"https://bancheck.tsunstudio.pw/bancheck?uid={uid}")
data = response.json()

print(f"Player: {data['nickname']}")
print(f"Banned: {data['is_banned']}")

JavaScript (Fetch)

javascript 
const uid = "2513698016";
fetch(`https://bancheck.tsunstudio.pw/bancheck?uid=${uid}`)
  .then(response => response.json())
  .then(data => {
    console.log(`Player: ${data.nickname}`);
    console.log(`Banned: ${data.is_banned}`);
  });
